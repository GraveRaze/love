# Romantic Personal Website

A multi-page, responsive personal website designed with a romantic theme. Perfect for a special gesture to a partner.

## Project Structure

- `index.html`: The main landing page with a hero section and "Next" button.
- `about.html`: The second page for your personal message.
- `styles.css`: Custom CSS for animations, colors, and layout (linked to Bootstrap).
- `scripts.js`: JavaScript for smooth navigation and effects.

## How to Run

1.  Navigate to the project folder: `d:/Didi/last_wish`
2.  Double-click `index.html` to open it in your default web browser.
3.  Click the "Next" button to navigate to the message page.

## Customization

-   **Message**: Open `about.html` in a text editor and change the text inside `class="message-text"` to your own personal message.
-   **Images**: You can add an `<img>` tag in `index.html` or `about.html` to include photos.

## Tech Stack

-   HTML5
-   CSS3 (+ Bootstrap 5)
-   Vanilla JavaScript
